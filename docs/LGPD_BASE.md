# LGPD Base (Modelo Inicial)

> Documento base para iniciar adequacao. Revisao juridica profissional e recomendada.

## Dados tratados
- Cadastro: nome, e-mail, telefone, CPF, data de nascimento.
- Operacionais: agendamentos, informacoes de atendimento.
- Tecnicos: logs de acesso, IP, user-agent.

## Finalidades
- Prestacao do servico de gestao medica.
- Seguranca da aplicacao e prevencao de fraude.
- Comunicacoes transacionais (ex.: redefinicao de senha).

## Bases legais (exemplo)
- Execucao de contrato.
- Legitime interesse para seguranca.
- Cumprimento de obrigacao legal/regulatoria quando aplicavel.

## Direitos do titular
- Confirmacao de tratamento.
- Acesso, correcao, portabilidade e eliminacao conforme lei.
- Revogacao de consentimento quando aplicavel.

## Retencao
- Dados mantidos pelo tempo necessario para operacao, requisitos legais e defesa de direitos.

## Seguranca
- Controle de acesso por perfil.
- Criptografia de senha.
- Logs e auditoria minima.
- Backup com retencao definida.

## Canal de atendimento
- Definir e-mail oficial para solicitacoes LGPD (ex.: privacidade@seu-dominio.com).
