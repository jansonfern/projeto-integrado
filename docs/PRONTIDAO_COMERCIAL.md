# Prontidao Comercial (Checklist)

## Produto
- [ ] Fluxo de cadastro/login/agenda sem erros
- [ ] Recuperacao de senha ativa
- [ ] Perfil e permissao por papel revisados
- [ ] Mensagens de erro amigaveis

## Operacao
- [ ] Deploy de producao documentado
- [ ] Backup diario com restauracao testada
- [ ] Monitoramento de erros ativo
- [ ] Processo de suporte definido

## Qualidade
- [ ] Testes automatizados no CI
- [ ] Regressao manual em release
- [ ] Checklist de aceite antes de publicar

## Juridico/LGPD
- [ ] Politica de Privacidade publicada
- [ ] Termos de Uso publicados
- [ ] Processo de atendimento a solicitacao do titular
- [ ] Controle de acesso a dados sensiveis

## Comercial
- [ ] Landing page com proposta clara
- [ ] Planos e precificacao definidos
- [ ] Contrato/SLA padrao
- [ ] Material de demo e onboarding
